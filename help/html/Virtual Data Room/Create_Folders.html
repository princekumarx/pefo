Creating Folders

Documents module gives you the flexibility to save documents in selected folder.

To create a custom folder

  1. Click on Add Folder button.

  2. Enter Folder Name (Mandatory) and Description.
  3. Click Save.

To create a custom sub folder :-

  1. Click on Add Sub Folder button.

  2. Select Folder (Mandatory) from drop-down for which sub folder needs to be created.
  3. Enter New Sub Folder Name (Mandatory).
  4. Click Save.


Note! Folders will be displayed to all users in your PE Front Office. If you do not want files to be displayed to other users, enable permissions(Eg: Private) from Sharing Access.
