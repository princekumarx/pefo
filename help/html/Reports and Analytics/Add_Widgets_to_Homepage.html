Add Widgets to Homepage

To add widgets to the Homepage
  1. Open the Home page and click on icon at the top.
  2. You will see a dropdown with 3 options -
    • Module
    • Notebook
    • Default Widgets
  3. Click Module to add existing modules like Contacts, Organizations, etc as widgets. Description of Module widgets is given later in this document.
  4. Click Notebook to add a notebook widget. This is similar to a sticky note.
  5. PE Front Office provides two Default Widgets - Pending Activities and Upcoming Activities.

Module Widgets

You can add the existing modules like Contacts, Organizations, etc as widgets on homepage for quick access. To add a Module Widget
  1. On Home screen, click icon at the top and click Module in the dropdown.
  2. You will see Add Module Widget window.
    3. Window Title Provide title of the widget.
    Show Select number of rows to be displayed in widget.
    Module Select the module.
    Filter by Select filter. Picklist values are 'Prospect Accounts' and 'New this Week'.
    Fields to Show Select fields of the module to show in widget. You can select maximum two fields.
  3. Fill the details and click Save.
