Portfolio Investments

This tab contains Existing Portfolio, Portfolio Actual/Realized Cashflow, Portfolio Expected Cashflow and Portfolio Valuation

.

Existing Portfolio shows
  1. Portfolio Company
  2. Sector
  3. Date of Investment
  4. Exit Status
  5. Investment Currency
  6. Total Investment (INR)
  7. Current Cost (INR)
  8. Total Realization (INR)
  9. Current Valuation (INR)
  10. Current Valuation Date
  11. Fund Allocation (%)


Portfolio Actual/Realized Cashflow shows following data populated from Portfolio module.

  1. Tran Date (yyyy-mm-dd)
  2. Portfolio Company
  3. Instrument Name
  4. Instrument Type
  5. Instrument CCY
  6. Transaction Type
  7. No of Instruments
  8. Transaction Amount (Inst CCY)
  9. Transaction Amount (Fund CCY)
  10. Price per Inst. Inst CCY)
  11. Price per Inst. (Fund CCY)


Portfolio Expected Cashflow shows following data populated from Portfolio module.

  1. Tran Date (yyyy-mm-dd)
  2. Porfolio Company Name
  3. Instrument Name
  4. Instrument Type
  5. Instrument CCY
  6. Transaction Type
  7. No of Instruments
  8. Transaction Amount (Inst CCY)
  9. Price per Inst. (Inst CCY)


Portfolio Valuation shows following data populated from Portfolio module.

  1. Valuation Date (yyyy-mm-dd)
  2. Porfolio Company Name
  3. Inst. Name
  4. Instrument Type
  5. Instrument CCY
  6. Valuation Method
  7. Health Status
  8. Valuation (Inst CCY)
