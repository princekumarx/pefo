Investors Drawdown / Distribution

This tab contains Drawdown / Distribution history and allows users to Import Drawdown Distribution, Units/Shares Summary and Download report in excel Portfolio Level - Investor Contribution and another tab Fund Level (underlying) Investor Contribution.

Drawdown / Distribution History shows
  1. Letter Date
  2. GL Date/Due Date
  3. Purpose/Heading
  4. Drawdown
  5. Drawdown %
  6. Distribution
  7. Distribution (%)
  8. Upload Type
  9. Remark

Download Report will download complete data in excel sheet as display on the screen.

Generate Template will download Template in excel and show how to import data from excel.

Import Drawdown Distribution will import data from excel and display on the screen.

Download link in Action will download imported Drawdown Distribution excel sheet for the record display on the screen.


You can add Drawdown / Distribution History using Import (Manual or Excel import) and Edit or Delete using links in the Action column.

View link will display complete detail of Drawdown or Distribution of a perticular record display on the screen.





Units/Shares Summary shows
  1. Notice Date
  2. Notice Type D/D
  3. Purpose/Heading
  4. Investor Type
  5. Investor ID
  6. Units Issued
  7. Units Redeemed
  8. Units Type
  9. Closing Balance



Portfolio Level - Investor Contribution shows
  1. Portfolio Name
  2. Portfolio Code
  3. Investor Name
  4. Type
  5. Fund CCY
  6. Investor Contribution
  7. Contribution %



Fund Level (underlying) Investor Contribution shows
  1. Fund Name
  2. Fund Type
  3. Letter Date
  4. Due Date
  5. Investor Name
  6. Type
  7. Fund CCY
  8. Investor Contribution
  9. Contribution %
