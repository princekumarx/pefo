Fund Investments

This tab contains Fund Invested Summary, Drawdown Distribution Cashflow, Units Allocation Summary Panel and Valuation Summary

.

Fund Invested Summary shows
  1. Fund Name
  2. Fund Type
  3. Fund Inv CCY
  4. Total Committed (USD)
  5. Contribution Amount (USD)
  6. Balance Committed (USD)
  7. Total Realizations (USD)
  8. Valuation Date
  9. Current Valuation (USD)


Drawdown Distribution Cashflow shows following data:

  1. Fund Name
  2. Letter Date
  3. GL Date/Due Date
  4. Purpose/Heading
  5. Transaction Type
  6. Transaction Category
  7. Portfolio Company/Fund Name
  8. Transaction CCY
  9. Transaction Amount
  10. Fund Transaction Date
  11. Fund Transaction Amount (USD)
  12. Fund Transaction Exchange Rate


Units Allocation Summary Panel shows following data :

  1. Fund Name
  2. Total No of Units Issued
  3. Total No of Units Redeemed
  4. Current No of Units
  5. Face Value
  6. Type of Document


Valuation Summary shows following data populated:

  1. Fund Name
  2. Valuation Date
  3. Valuation CCY
  4. Valuation Amount
  5. Valuation Adjustment
  6. Valuation Amount (Adjusted)
  7. Health Status
  8. Notes
