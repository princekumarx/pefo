Custom Settings

Deal Stage Workflow

PE Front Office allows you to Add new deal stage and Edit or Delete existing deal stage. When a new deal stage is created, deal stage created user details display in Added column.

If a deal stage is used in Deal Stage Workflow panel of a Deal Pipeline then, this deal stage becomes Archive and can not be edit or delete from CustomSettings. If a Deal stage is Unarchive from CustomSettings then this is not visible in Deal Stage Workflow panel of a Deal Pipeline for newly created deals.
