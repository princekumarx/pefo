Master Data Management

PE Front Office allows you to manage Master Data and Transaction Type data.

This can be done by selecting Field Type from drop-down which are Master Data and Transaction Type.

Display Picklist value. If selected Master data Field from drop-down, if its Active column value is Yes then display in Picklist and if active column is No then can not display in Picklist.

Add or Edit value of selected field from dropdown.

Select Transaction Type Fields from drop-down (Fund cost, Transaction category primary, Transaction category secondary).

Add or Edit values of different transaction types.
