Additional Information

This tab contains information about Funds, Co-Investors, Investment Team, External Team and Deal Stage History.

Funds list view contains following information
  1. Name of the Fund.
  2. Currency of the Fund.
  3. Percentage of proposed stake to be invested by respective Fund.
You can add Funds by clicking the Add link in top right and Edit or Delete using links in the Actions column.

Co-Investors list view contains following details of the Co-Investors Team members.

  1. Investor Name
  2. Investor Type
  3. Interest Level
  4. Status
  5. Likely Investment
  6. Initial Contact
  7. Term Sheet Sent
  8. DD Access
  9. Legals Sent
  10. Closure
  11. Notes


Investment Team list view contains following details of the Investment Team members.

  1. Name
  2. Designation
  3. Role

You can add members in the Investment Team by clicking the Add link in top right and Edit or Delete using links in the Actions column.


External Team list view shows following details of the Team associated with the Deal.

  1. Name
  2. Organization Name
  3. Role (Contact's role in Deal)
You can add External Team by clicking the Add link in top right and Edit or Delete using links in the Actions column.

Deal Stage History shows all the stages through which Deal has passed and the current stage. List view contains following information

  1. Deal Stage.
  2. Date when transition was made to respective deal stage.
  3. Name of the User who changed the stage.
