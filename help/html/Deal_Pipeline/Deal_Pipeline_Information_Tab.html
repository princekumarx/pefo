Workflow


Workflow is to track various activities mapped to different stages of the deal.
  1. Deal Stage Stage of the deal at which an activity is expected to be completed.
  2. Activity Description of the activity
  3. Folder Clickable folder name where the documents associated with an activity should be uploaded
  4. M/O Indicates if an activity is Mandatory or Optional. Deal cannot move to the next stage unless mandatory activity for that stage has been done.
  5. Completed Shows if an activity has been done. Once marked as completed workflow entry cannot be edited.
  6. Notes User can enter any notes related to that activity for others to look at.
