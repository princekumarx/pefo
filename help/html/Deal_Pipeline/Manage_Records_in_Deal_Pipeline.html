Manage Deal Pipeline

List of operations you can perform on records in Deal Pipeline module

1. Create Custom filters to classify records as per your requirements. You can limit your search to selected columns and search criteria.
2. Search desired records.
3. Go to top left of the List view to perform following bulk operations
Click Delete button to delete selected record(s) in List view.
Click Mass Edit button to edit selected record(s) in List view.

List of operations you can perform on detail view of Deal Pipeline record


1. Click Edit to modify existing content.
3. Click Delete to delete the existing Deal Pipeline record.
4. Click Duplicate to clone existing Deal Pipeline record.
5. Click Add Document to attach a document to existing Deal Pipeline record.
6. Click View History to to view history of changes made on Deal Pipeline record.
7. Click Add Event to schedule a Call or Meeting and Select Add To Do to schedule a task.
8. Apply Currency Conversion.
9. View Reports.
