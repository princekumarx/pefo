Quick Create

You can enter basic details of Deal Pipeline through Quick create. This might be helpful if you are running out of time.
Steps to create Deal Pipeline through Quick create
  1. Click Quick Create drop-down at the top right.
  2. Choose New Deal Pipeline.
  3. In the Create Deal Pipeline view, fill in the details.
  4. Click Save.
