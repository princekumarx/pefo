Currency Conversion

Currency conversion helps you to view following in different currencies.

  1. Deal Size (m)
  2. Debt (m)
  3. Enterprise Value (m)
  4. Equity Value (m)

To apply currency conversion
  1. In detailed view of Deal Pipeline, click on Apply Currency Conversion on the right.
  2. You will see a small window containing Deal Pipeline parameters and a target currency dropdown.
  3. Choose the currency from the dropdown.
  4. Deal Pipeline's information in the selected currency will populate automatically.
