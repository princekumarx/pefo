Install

Follow few simple steps to install PE Front Office plugin for Outlook

  1. Close Outlook if already running
  2. Open file location, double click on the file to execute
  3. Click Next
  4. Select Installation Folder. Click Next
  5. Confirm Installation. Click Next
  6. Installation Complete. Click Close
