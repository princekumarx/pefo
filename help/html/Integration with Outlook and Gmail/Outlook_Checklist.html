Outlook Checklist

Depending upon the version of Microsoft Outlook plugin, you'll need to install the corresponding PE Front Office Outlook plugin. To check the Outlook version

  1. Open Microsoft Outlook
  2. Click File -> Help


PE Front Office 32 bit (PEFrontOfficeOutlookPlugin-1.0-X86) Outlook Plugin supports

  1. Microsoft Outlook 2010 32 bit
  2. Microsoft Outlook 2013 32 bit

PE Front Office 64 bit (PEFrontOfficeOutlookPlugin-1.0-X64) Outlook Plugin supports

  1. Microsoft Outlook 2010 64 bit
  2. Microsoft Outlook 2013 64 bit
