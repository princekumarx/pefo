Tool Tip Management helps you to manage tool tip information in Documents module.


Manage Tool Tip Information

  1. Click Settings Icon at the top.
  2. Click Tool Tip Management.
  3. Select the field from Select Field picklist for which you want to manage tool tip information. Attributes of the selected field will appear with check boxes next to each attribute.
  4. Enable the checkbox next to the information you want to show in the tool tip.
  5. Similarly, disable the checkbox for the information which was enabled earlier and you want to hide in the tool tip.
  6. Click Save.


Note! All checkboxes are disabled by default.


View Tool Tip Information

Tool Tip Information will automatically reflect on mouse hover on the field.
