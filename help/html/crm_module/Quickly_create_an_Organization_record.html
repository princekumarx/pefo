Quick Create

You can enter basic details of Organization record through Quick create. This might be helpful if you are running out of time.

Steps to create Organization records through Quick create

  1. Click Quick Create drop-down at the top right corner.
  2. Choose New Organization.
  3. In the Create Organization view, fill in the Organization details.
  4. Click Save.
