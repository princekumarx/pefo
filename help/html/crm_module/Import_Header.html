In the step 4 of import process, you'll have to map fields in .csv file with fields in your Sumantra CRM account (See image below). The fields from your .csv file header are displayed on the left and fields in Sumantra CRM to right. Now your job is to map these fields.



That's the reason why having a header in your .csv file is important and handy. Also, if your file has a header, please do not forget to enable the checkbox Has Header in the step-2 of import process. It's good to have a header in your .csv file but not mandatory. If your file doesn't has a header, please disable the checkbox Has Header in the step-2 of import process. However, if your file doesn't have a header, the fields in the first row are displayed during mapping.
