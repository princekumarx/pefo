Funds Administration

Enter Drawdown/Distribution


Drawdown / Distribution History shows
  1. Letter Date
  2. GL Date/Due Date
  3. Purpose/Heading
  4. Drawdown
  5. Drawdown %
  6. Distribution
  7. Distribution (%)
  8. Upload Type
  9. Remark

Download Report will download complete data in excel sheet as display on the screen.

Generate Template will download Template in excel and show how to import data from excel.

Import Drawdown Distribution will import data from excel and display on the screen.

Download link in Action will download imported Drawdown Distribution excel sheet for the record display on the screen.


You can add Drawdown / Distribution History using Import (Manual or Excel import) and Edit or Delete using links in the Action column.

View link will display complete detail of Drawdown or Distribution of a perticular record display on the screen.
