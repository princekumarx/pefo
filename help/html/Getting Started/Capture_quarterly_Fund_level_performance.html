Capture quarterly Fund level performance

This tab contains information about Portfolio Investment Performance Summary (Qtr), Investor Drawdown Distribution Summary (Qtr), Fund Cost.

Portfolio Investment Performance Summary (Qtr) shows
  1. Date
  2. Current
    • Gross
      • IRR(%)
      • MoC
    • Net
      • IRR(%)
      • MoC
  3. Expected
    • Gross
      • IRR(%)
      • MoC
  4. Total Investment (Fund CCY)
  5. Total Realization (Fund CCY)
  6. Current valuation (Fund CCY)
  7. Assets (Fund CCY)
  8. Liabilities (Fund CCY)
  9. NW/NAV
  10. Reported to Investor

Valuation should be define for Underline Portfolio Company(s) for reporting date before adding entry in this panel.

Date wise you can add Portfolio Investment Performance Summary (Qtr) by clicking on Add link in top right and Edit or Delete using links in the Action column.



Investor Drawdown Distribution Summary (Qtr) shows
  1. Date
  2. Total Committed
  3. Drawdown
  4. Distribution
  5. DPI
  6. TVPI
  7. Fund Gross IRR
  8. Fund Net IRR
  9. Units Issued
  10. Units Redeemed

Entry should be define for reporting date in "Portfolio Investment Performance Summary (Qtr)" panel before adding entry in this panel.


Fund Cost

Fund Cost shows the internal expenses of the fund for example - bank charges, travel expenses, etc. List view contains following details.

  1. Date
  2. Portfolio Company
  3. Transaction Type
  4. Transaction Currency
  5. Transaction Amount
  6. Fund CCY Amount

You can add new expenses by clicking on Add link in top right and Edit or Delete using links in the Action column.
