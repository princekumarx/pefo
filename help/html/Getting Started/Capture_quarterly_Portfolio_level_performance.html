IRR and MoC History list view contains following details
  1. Date
  2. IC Case IRR %
  3. Expected Gross IRR %
  4. Gross IRR %
  5. IC Case MoC
  6. Expected Gross MoC
  7. Gross MoC
  8. Reported to Investor - Yes/No
You can add IRR and MoC History by clicking the Add link in top right and Edit or Delete using links in the Action column.
