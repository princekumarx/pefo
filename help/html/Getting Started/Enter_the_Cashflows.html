Amortization Schedule Panel

Select instrument from the dropdown and data corresponding to selected instrument will be populated. List view shows following details of the selected instrument
  1. End Date
  2. No. of Inst Purchased
  3. Bonds Held
  4. Principal Held
  5. Purchase Discount
  6. Acquisition Amount
  7. Acquisition Fees (to be paid)
  8. Interest Paid on Acquisition
  9. Principal Repayment (to be recd)
  10. Return of Capital (Security Receipts)
  11. Redemption Premium (after WHT)
  12. Cash Interest (after WHT)
  13. Interest Accrued/PIK
  14. PIK Bonds
  15. PIK Interest (after WHT)
  16. Fee Income (after WHT)
  17. Equity Upside (after WHT)
  18. Dividend
  19. No. of Inst Sold

You can aslo perform multiple operations on the selected instrument like
  1. Generate/Download Amortization Schedule : Automatic or Manula
  2. Import Amortization Schedule


    3. Import downloaded Amortization sheet by click on Choose File and import is done only when Is Validate should be Yes in sheet.
  3. View Amortization Schedule History
  4. Generate Report(Amortization Schedule vs Payments
  5. Close Amortization Schedule
    6. If Amortization is Closed for an instrument then all transaction should display in Gray color and user unable to make payment against any transaction i.e. while clicking on Viewpayment from Action, Add link hide from add payment popup.
You can get Payment details by clicking on Cashflow link next to each record in the list view.

Add Payment from View Payment link.



View Amortization Schedule History : You can Download Amortization Sheet imported by user on different dates.


Actual Cashflow Panel

Select instrument from the dropdown and data corresponding to selected instrument will be populated. List view shows following details of the selected instrument
  1. Tran Date
  2. Inst. Name
  3. Instrument Type
  4. Investment CCY
  5. Fund/Investor CCY
  6. Tran Type
  7. Transaction Description
  8. No. of Instruments
  9. Transaction Amount (Inv CCY)
  10. Transaction Amount (Fund CCY)
  11. Exchange Rate
  12. Price per Inst. (Inv CCY)
  13. Price per Inst. (Fund CCY)
  14. Notes
  15. TDS
You can add records by clicking the Add link in top right and Edit or Delete using links in the Action column.

Download Template will download Template in excel and show how to import data from excel.

Import Actual Data will import data from excel and display on the screen.

Export To XLS will download data in excel sheet as display on screen.
