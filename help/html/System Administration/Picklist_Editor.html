Sharing Profiles

As the name itself says, Picklist is a drop-down field with list of options available, within which, only one option can be selected. For instance, Job Title in Contacts Module. Picklist Editor is used to configure picklist values in different modules. It enables you to add new values, rename, delete or rearrange existing picklist values. Picklist values vary across roles â€” users under a role can only access picklist values assigned to that role.


Get Started

You can create a custom picklist field with desired set values on a selected module. While creating records, select a value from the picklist to have it automatically inserted in the field.


Working with Picklist Fields

You can perform a wide variety of operations on picklist fields like adding new values to existing picklist field, modifying, deleting, rearranging existing values of a picklist field. Additionally, you can also assign roles on selected picklist values.


Add Picklist Item

  1. Click gear icon in the top right.
  2. Click CRM Settings.
  3. Click on Picklist Editor under Studio.
  4. Select Module.
  5. Select Picklist.
  6. Click on Add Item. You will see Add Picklist Value pop up window.


    7. The pop up window will show all the existing values for the selected Module and Picklist.
  7. Select Role.
  8. Enter the new Value for Picklist..
  9. Click Save.


Assign Picklist Item

  1. Open Picklist Editor.
  2. Select Module.
  3. Select Role.
  4. You will see all the Picklists for the selected module.
  5. Click on Assign button for any Picklist. You will see Assign Picklist Value pop up.



    6. The pop up window will show all the values in picklist and assigned values to the selected Role in selected Module.
  6. Assign values using arrows.
  7. Click Save.


Rename Picklist Item

  1. Open Picklist Editor.
  2. Select Module.
  3. Select Picklist.
  4. Click on Rename Item button. You will see Edit Picklist Value pop up window.



    5. The pop up window will show all the existing values for the selected Module and Picklist.
  5. Select the Value we want to rename from the list.
  6. Enter the new Value in 'Replace with' field.
  7. Click Save.

Delete Picklist Item

  1. Open Picklist Editor.
  2. Select Module.
  3. Select Picklist.
  4. Click on Delete Item button. You will see Delete Picklist Value pop up window.



    5. The pop up window will show all the existing values for the selected Module and Picklist.
  5. Select the Value we want to delete from the list.
  6. Select the Existing Value which will replace the deleted value in the records.
  7. Click Save.
