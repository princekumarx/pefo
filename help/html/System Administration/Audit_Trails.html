Audit Trails

PE Front Office allows you to Enable Audit Trail and View Audit Trail Report for a selected user.


Here's how to manage the Audit Trails

  1. Click gear icon in the top right.
  2. Click CRM Settings.
  3. Click Audit Trail under Users & Access Management.
  4. You will see Audit Trail window. You can provide following details
    • You need to have Enable Audit Trail check box option enabled to track the actions performed by users on records.
    • Select a user from Select User picklist and click on View Audit Trail button.


    • Instantly, a popup appears showing the Audit trails under five columns namely Module, displays the module on which action was performed; Action, displays the action that was performed; Record ID, displays the record ID and Action Date, displays the date when action was performed.
