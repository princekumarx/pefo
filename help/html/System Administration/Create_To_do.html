You can automatically create to dos based on workflow. The following workflow is supposed to create a todo when Deal Stage becomes Screened.


1. Select Deal Pipeline in Select Module and my condition will be Until the first time the condition is true



2. The goal is to create todo only when deal stage is screened; so, my condition will be Deal Stage is Screened. If condition is met, workflow will be triggered on the record. Once executed, workflow doesn't trigger action on that record again.



3. Click Save


4. Click on New Task button and select Create To do. Provide Task Title; this is only for your reference. Todo Name, Description, Status, Priority, Time, Due Date, Send Notification are displayed in the event that is created when workflow is executed. Due Date cannot be defined statically. They are defined basing upon either created time or modified time of the parent record (Deal Pipeline in our example).



5. Click save
