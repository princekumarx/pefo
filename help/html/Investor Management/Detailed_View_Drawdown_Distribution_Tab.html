Detailed View - Drawdown / Distribution Tab

This tab shows Drawdown / Distribution of the Fund(s) in which Investor has invested. The gridview contains following fields


  1. Fund Name
  2. Letter Date
  3. GL Date/Due Date
  4. Purpose/Heading
  5. Transaction Type
  6. Transaction Category Primary
  7. Transaction CCY
  8. Transaction Amount
  9. Fund Currency Amount
  10. Investment Type
  11. Investment

By selecting Fund from drop-down that particular Fund data display on the screen. If user select ALL from drop-down then all funds data display on the screen.


Export To XLS will download data in excel sheet as display on screen.
