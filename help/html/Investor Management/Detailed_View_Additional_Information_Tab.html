Detailed View - Additional Information Tab

This tab contains information of LP team members, Advisors and Co-Investment Panel.


LP Team

LP Team list view contains following details of the team members of the Investor

  1. Name
  2. Role
  3. Primary Contact
  4. Designation
  5. Email
  6. Phone
  7. Mailing List
You can add LP Team members by clicking the Add link in top right and Edit or Delete using links in the Actions column.


Notes!

You can only add your Contacts as LP Team members.


Advisors

This is the list of Investor's Advisors. List view shows following details

  1. Name
  2. Role
  3. Designation
  4. Email
  5. Phone
You can add Advisors by clicking the Add link in top right and Edit or Delete using links in the Actions column.



Notes!

You can only add your Contacts as Advisors.


Investments in Other Funds

This is the list of Investments in Other Funds. List view shows following details

  1. Fund Name
  2. Year of Investment
  3. Currency
  4. Amount
  5. Notes
You can add Advisors by clicking the Add link in top right and Edit or Delete using links in the Actions column.
